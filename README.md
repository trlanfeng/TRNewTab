# TRNewTab
# 幻蓝新标签页

幻蓝新标签页，用于替换谷歌浏览器中的默认新标签页，支持自定义搜索、防止搜索劫持。

免费、简洁、快速、易用、可同步、无广告。

## 开发缘由

厌倦了一大堆功能繁多、花里胡哨的新标签页，只想简简单单的有个像Opera一样的快速拨号。

厌倦了一大堆快速拨号里各种搜索劫持（在某些网络环境下会导致多次刷新），各种广告。

于是自己开发了一个简单、可同步的快速拨号插件。
